DJMAX Style Rhythm Game

PixiJS와 Web Audio API를 활용한 DJMAX 스타일의 리듬 게임입니다. 에디터에서 차트를 제작하고, 플레이어에서 게임을 즐기세요!

Chart Editor

MP3 파일을 업로드하여 나만의 리듬 게임 차트를 제작하세요. BPM 자동 감지와 실시간 노트 배치 기능을 제공합니다.

Game Player

제작한 차트를 플레이하세요! 5키 시스템 (D, F, Space, J, K)으로 완벽한 타이밍에 노트를 터치하세요.

주요 기능

PixiJS v8 렌더링

고성능 WebGL 기반 2D 렌더링으로 부드러운 게임 플레이를 제공합니다.

Web Audio API

정밀한 오디오 타이밍과 BPM 자동 감지 기능을 제공합니다.

5키 시스템

접근성이 좋은 5레인 (D, F, Space, J, K) 시스템을 사용합니다.

반응형 디자인

모바일과 데스크톱 모두에서 최적화된 게임 플레이를 경험하세요.

판정 시스템

PERFECT, GREAT, GOOD, MISS의 정확한 타이밍 판정을 제공합니다.

차트 저장/로드

제작한 차트를 JSON 형식으로 저장하고 불러올 수 있습니다.

Built with PixiJS v8 and Web Audio API
